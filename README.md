# pmxtrader

Prediction market trading platform with PMXT, MCP, and agentic capabilities.

## Overview

`pmxtrader` is a custom-built prediction market trading system centered around PMXT. It focuses on clean market analysis, CLI tooling, and future support for both terminal and web interfaces, with strong emphasis on MCP and agentic integration.

## Structure

- `pmxt/` — Core PMXT integration layer
  - `client/` — PMXT API client and wrappers
  - `analysis/` — Structured market analysis and reporting
  - `mcp/` — MCP tools for AI agents
  - `cli/` — Custom CLI commands and terminal dashboard
- `apps/` — Applications (dashboard, agents, CLI)
- `packages/` — Shared components and utilities
- `tools/` — Paper trading, backtesting, and utilities
- `docs/` — Architecture and project decisions

## Status

Early development. Focused on building a clean terminal-based analysis experience first.
